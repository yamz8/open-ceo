Run through a pre-send checklist before sending your investor update.

## Purpose

Catch common mistakes and ensure your update is polished before hitting send.

## Checklist Categories

### 1. Recipients
- [ ] All investors included?
- [ ] Board members on the list?
- [ ] Any new investors to add?
- [ ] Using BCC for large lists?
- [ ] Correct email addresses?

### 2. Content Quality
- [ ] Subject line is clear and dated?
- [ ] TL;DR at the top?
- [ ] Metrics are accurate and current?
- [ ] Highlights are specific (not vague)?
- [ ] Challenges addressed honestly?
- [ ] Clear asks included?

### 3. Tone Check
- [ ] Professional but not stiff?
- [ ] Confident but not arrogant?
- [ ] Transparent about difficulties?
- [ ] Respectful of reader's time?

### 4. Data Accuracy
- [ ] ARR/MRR figures verified?
- [ ] Growth calculations correct?
- [ ] Runway reflects latest burn?
- [ ] Customer counts accurate?
- [ ] No outdated numbers from drafts?

### 5. Formatting
- [ ] Easy to scan quickly?
- [ ] Consistent formatting?
- [ ] Links working?
- [ ] No typos in key names/numbers?
- [ ] Mobile-friendly length?

### 6. Timing
- [ ] Sent at reasonable hour?
- [ ] Not conflicting with major news?
- [ ] Consistent with your update cadence?

## Interactive Walkthrough

Walk through each category with the user:

1. Ask if they have a draft to review
2. If yes, read the draft and check automatically where possible
3. For each category, confirm items are addressed
4. Flag any concerns found
5. Provide final go/no-go recommendation

## Output

Provide a summary:

```
✅ Pre-Send Checklist Complete

Passed: 18/20 items
Warnings: 2

⚠️ Warnings:
- Runway figure may be outdated (last updated 3 weeks ago)
- No specific ask included in this update

Recommendation: Address warnings before sending, or proceed if intentional.
```

## Common Issues to Flag

- Stale metrics (more than 2 weeks old)
- Missing TL;DR section
- Vague highlights without specifics
- No asks (missed opportunity)
- Overly long update (>500 words)
- Defensive tone about challenges
