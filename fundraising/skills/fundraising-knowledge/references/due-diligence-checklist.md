# Due Diligence Checklist by Stage

## Pre-Seed / Seed DD Checklist

Due diligence at early stages is lighter but still important to prepare.

### Corporate (Essential)
- [ ] Certificate of Incorporation
- [ ] Bylaws
- [ ] Cap table (use Carta, Pulley, or spreadsheet)
- [ ] List of all shareholders and option holders
- [ ] Stock option plan document
- [ ] 83(b) elections for founders

### Founder Background
- [ ] Founder LinkedIn profiles
- [ ] Background check consent
- [ ] Prior company history
- [ ] Reference contacts (3-5 people)

### Product/Tech
- [ ] Product demo or walkthrough
- [ ] Technical architecture overview (1-pager)
- [ ] IP assignment agreements from founders
- [ ] Any patents or patent applications

### Financial
- [ ] Bank statements (last 3 months)
- [ ] Current burn rate
- [ ] Simple financial model (12-month projection)

### Legal
- [ ] Any existing investor agreements (SAFEs, notes)
- [ ] Key contracts (if any)
- [ ] Any litigation or threatened claims

## Series A DD Checklist

More comprehensive diligence expected at Series A.

### Corporate Documents
- [ ] Certificate of Incorporation (restated)
- [ ] Bylaws (current)
- [ ] All board meeting minutes
- [ ] All shareholder meeting minutes
- [ ] Cap table (fully diluted, Carta export preferred)
- [ ] Stock option plan and all grants
- [ ] All outstanding SAFEs/notes with conversion terms
- [ ] List of all subsidiaries (if any)
- [ ] Good standing certificates

### Financial
- [ ] Monthly P&L (last 24 months)
- [ ] Balance sheet (current)
- [ ] Cash flow statement
- [ ] Bank statements (last 6 months)
- [ ] Financial model (3-year projection)
- [ ] Accounts receivable aging
- [ ] Accounts payable summary
- [ ] Revenue by customer (top 20)
- [ ] MRR/ARR breakdown and cohort analysis

### Commercial/Customers
- [ ] Customer list with contract values
- [ ] Top 10 customer contracts
- [ ] Standard customer agreement template
- [ ] Churn analysis by cohort
- [ ] NPS or customer satisfaction data
- [ ] Sales pipeline summary
- [ ] Customer references (5-10 for calls)

### Product/Technology
- [ ] Product roadmap
- [ ] Technical architecture document
- [ ] Security practices overview
- [ ] SOC 2 status (or timeline)
- [ ] Data privacy compliance (GDPR, CCPA)
- [ ] Third-party service dependencies
- [ ] Open source usage policy
- [ ] Uptime/reliability metrics

### Intellectual Property
- [ ] List of all IP (patents, trademarks, domains)
- [ ] Patent applications
- [ ] IP assignment agreements (all employees/contractors)
- [ ] CIIAA agreements signed
- [ ] Any IP disputes or claims

### Team/HR
- [ ] Org chart
- [ ] Employee list with titles, start dates, comp
- [ ] Key employee contracts
- [ ] Offer letter templates
- [ ] Employee handbook
- [ ] Benefits summary
- [ ] Any employment disputes
- [ ] Contractor agreements (key contractors)

### Legal
- [ ] All material contracts
- [ ] Vendor agreements (>$50K/year)
- [ ] Partnership agreements
- [ ] Any litigation (pending or threatened)
- [ ] Insurance policies (D&O, E&O, cyber)
- [ ] Regulatory compliance status

---

## Series B+ DD Checklist

Comprehensive diligence with financial and operational deep dives.

### Everything from Series A, plus:

### Financial Deep Dive
- [ ] Audited financials (or audit-ready)
- [ ] Unit economics by segment/product
- [ ] CAC/LTV analysis by channel
- [ ] Revenue recognition policy
- [ ] Deferred revenue schedule
- [ ] Budget vs actuals (last 2 years)
- [ ] Scenario analysis (bear/base/bull)
- [ ] Path to profitability analysis

### Go-to-Market
- [ ] Sales process documentation
- [ ] Win/loss analysis
- [ ] Competitive positioning
- [ ] Marketing spend and ROI by channel
- [ ] Sales rep productivity metrics
- [ ] Partner/channel agreements

### Operations
- [ ] Key vendor contracts and dependencies
- [ ] Infrastructure costs and scaling plan
- [ ] Support metrics (response time, CSAT)
- [ ] Operational KPIs dashboard

### Compliance/Risk
- [ ] SOC 2 Type II report
- [ ] Penetration test results
- [ ] Data processing agreements
- [ ] International compliance (if applicable)
- [ ] Risk register

### Data Room Best Practices

1. **Use a proper data room**: Dropbox, Google Drive, or dedicated (Ansarada, Datasite)
2. **Name files clearly**: `YYYY-MM-DD_DocumentType_Description.pdf`
3. **Keep current**: Update documents as things change
4. **Track access**: Know who viewed what
5. **Watermark sensitive docs**: Especially financials

## Timeline Expectations

| Stage | DD Duration | Common Delays |
|-------|-------------|---------------|
| Pre-seed | 1-2 weeks | Cap table issues |
| Seed | 2-4 weeks | Missing IP assignments |
| Series A | 4-6 weeks | Financial reconciliation |
| Series B+ | 6-8 weeks | Legal/compliance issues |

### Common DD Killers

1. **Cap table mess**: Unclear ownership, missing 83(b)s
2. **IP issues**: No assignments, founder created IP elsewhere
3. **Customer concentration**: >30% revenue from one customer
4. **Legal disputes**: Undisclosed litigation
5. **Financial irregularities**: Revenue recognition issues
