# Pitch Deck Templates

## Sequoia Format - Slide by Slide

### Slide 1: Company Purpose
**Goal**: Capture attention in one sentence

**Template**:
```
[Company Name]
[One-sentence description of what you do]
[Tagline or mission statement]
```

**Example**:
```
Acme Corp
AI-powered inventory management for SMB retailers
"Never run out of stock again"
```

### Slide 2: Problem
**Goal**: Make the pain visceral and relatable

**Template**:
- State the problem in customer's words
- Quantify the pain (cost, time, frustration)
- Show who experiences this problem
- Explain why existing solutions fail

**Content structure**:
```
THE PROBLEM
[Customer quote or pain statement]

- [Stat 1]: [Impact metric]
- [Stat 2]: [Scale of problem]
- [Stat 3]: [Cost/consequence]

Current solutions fail because:
[Reason 1]
[Reason 2]
```

### Slide 3: Solution
**Goal**: Show your product solves the problem elegantly

**Template**:
```
THE SOLUTION
[Product name]: [What it does in one sentence]

Key Features:
1. [Feature 1] → [Benefit]
2. [Feature 2] → [Benefit]
3. [Feature 3] → [Benefit]

[Screenshot or product visual]
```

### Slide 4: Why Now
**Goal**: Create urgency and explain market timing

**Template**:
```
WHY NOW

Market Shifts:
- [Trend 1]: [Data point]
- [Trend 2]: [Data point]
- [Trend 3]: [Data point]

Technology Enablers:
- [Tech shift enabling your solution]

Regulatory/Cultural Changes:
- [Recent changes creating opportunity]
```

### Slide 5: Market Size
**Goal**: Show a large, growing, accessible market

**Template**:
```
MARKET OPPORTUNITY

TAM: $[X]B - [Total addressable market definition]
SAM: $[X]B - [Serviceable addressable market]
SOM: $[X]M - [Serviceable obtainable market in 3-5 years]

Market Growth: [X]% CAGR
Source: [Credible source]
```

**Bottoms-up calculation**:
```
[# of potential customers] × [Annual contract value] = $[X]
```

### Slide 6: Competition
**Goal**: Show you understand the landscape and have differentiation

**Template**:
Use 2x2 matrix with axes relevant to your differentiation

```
COMPETITIVE LANDSCAPE

        [Axis 1: Your strength]
              ↑
              |  [Competitor A]    [YOU]
              |
              |  [Competitor B]    [Competitor C]
              +----------------------→
                        [Axis 2: Your strength]

Key Differentiators:
1. [Unique advantage 1]
2. [Unique advantage 2]
3. [Unique advantage 3]
```

### Slide 7: Product
**Goal**: Demonstrate product with visuals

**Template**:
```
PRODUCT

[Screenshot/demo video embed]

How it works:
1. [Step 1]
2. [Step 2]
3. [Step 3]

Key integrations: [Logo row]
```

### Slide 8: Business Model
**Goal**: Show how you make money and unit economics

**Template**:
```
BUSINESS MODEL

Revenue Model: [Subscription/Usage/Transaction]

Pricing:
- [Tier 1]: $[X]/mo - [Features]
- [Tier 2]: $[X]/mo - [Features]
- [Enterprise]: Custom

Unit Economics:
- ACV: $[X]
- CAC: $[X]
- LTV: $[X]
- LTV:CAC: [X]:1
- Payback: [X] months
```

### Slide 9: Traction
**Goal**: Prove product-market fit with metrics

**Template**:
```
TRACTION

[Key metric graph showing growth]

Highlights:
- ARR: $[X]K → $[X]K ([X]% growth)
- Customers: [X] → [X]
- NRR: [X]%
- Logo retention: [X]%

Notable customers: [Logo row]
```

### Slide 10: Team
**Goal**: Show why this team will win

**Template**:
```
TEAM

[Photo] [Name] - CEO
[Relevant experience: Company, role, achievement]

[Photo] [Name] - CTO
[Relevant experience: Company, role, achievement]

[Photo] [Name] - [Role]
[Relevant experience]

Advisors: [Notable names]
```

### Slide 11: Financials
**Goal**: Show path to scale

**Template**:
```
FINANCIALS

         2024    2025    2026
Revenue  $[X]K   $[X]M   $[X]M
Growth   -       [X]%    [X]%
Burn     $[X]K   $[X]K   $[X]K
Runway   [X]mo   [X]mo   [X]mo
```

### Slide 12: Ask
**Goal**: Clear ask with use of funds

**Template**:
```
THE ASK

Raising: $[X]M [Seed/Series A]

Use of Funds:
- [X]% Engineering: [Specific goal]
- [X]% Sales/Marketing: [Specific goal]
- [X]% Operations: [Specific goal]

Milestones (18 months):
- [Milestone 1]
- [Milestone 2]
- [Milestone 3]
```

## YC Format - Slide by Slide

### Slide 1: What Do You Do?
```
[Company Name]

[One clear sentence explaining what you do]

Example: "Stripe for African payments"
Example: "Airbnb for office space"
```

### Slide 2: Problem
```
THE PROBLEM

[State problem simply]

[X]% of [target customers] struggle with [problem]
This costs them $[X] per [time period]
```

### Slide 3: Solution
```
[Product Name]

[Screenshot]

[One sentence on how it works]
```

### Slide 4: Traction
```
TRACTION

[Single compelling metric graph]

- [Key metric 1]: [Value]
- [Key metric 2]: [Value]
- Growth: [X]% MoM
```

### Slide 5: Market
```
MARKET

$[X]B market growing [X]% annually

[X]M potential customers × $[X] ACV = $[X]B
```

### Slide 6: Team
```
TEAM

[Founder 1]: [One-line credential]
[Founder 2]: [One-line credential]

Why us: [Unique insight or unfair advantage]
```

### Slide 7: Ask
```
RAISING $[X]

To achieve:
- [Milestone 1]
- [Milestone 2]
```

## a16z Format - Slide by Slide

The a16z format is more comprehensive, suitable for Series A+. Key additions:

### Mission/Vision Slide
```
OUR MISSION

[Bold, ambitious statement about changing the world]

[Visual representing the vision]
```

### Go-to-Market Slide
```
GO-TO-MARKET

Current Motion:
- [Channel 1]: [Performance]
- [Channel 2]: [Performance]

Expansion Strategy:
1. [Land strategy]
2. [Expand strategy]
3. [Scale strategy]

Sales Cycle: [X] days
Win Rate: [X]%
```

### Category Dynamics Slide
```
CATEGORY EVOLUTION

[Timeline or evolution visual]

Phase 1: [Past state]
Phase 2: [Current state]
Phase 3: [Future state - where you're positioned]

Category creation opportunity: [Explanation]
```
